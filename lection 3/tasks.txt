Задача 0
Напишите функцию, которая генерирует список из N элементов, состоящий из случайных целых чисел от 0 до 9.

Задача 1
Напишите функцию, которая принимает список и выводит все элементы случайного списка, которые меньше 5. Выведите -1, если таких нет.

Задача 2
Сгенерируйте два списка. Напишите функцию, которая выводит список общих для этих двух списков элементов. Выведите -1, если таких нет.

Задача 3
Нужно вывести первые n строк треугольника Паскаля. 
В этом треугольнике на вершине и по бокам стоят единицы, а каждое число внутри равно сумме двух расположенных над ним чисел.

Задача 4
Напишите функцию, которая проверяет является ли строка палиндромом. 
Палиндром — это слово или фраза, которые одинаково читаются слева направо и справа налево.

Задача 5
Напишите функцию, которая принимает целое число секунд и возращет значение в виде дни:часы:минуты:секунды.

Задача 6
Напишите функцию, которая принимает имя файла и выводит его расширение. Если расширение у файла определить невозможно, выведите -1.

Задача 7
Напишите функцию, которая принимает целое число n и вычисляет n + nn + nnn.

Задача 8
Напишите функцию, которая выводит чётные числа из заданного списка и останавливается, если встречает число 42.

Задача 9
Напишите функцию, которая принимает два списка и выводит все элементы первого, которых нет во втором. Выведите -1, если таких нет.

Задача 10
Напишите функцию, которая складывает цифры целого числа.

Задача 11
Напишите функцию, которая вычисляет сколько раз символ встречается в строке.

Задача 12
С помощью анонимной функции извлеките из списка числа, делимые на 15.

Задача 13
Напишите функцию, которая проверяет все ли числа в последовательности уникальны.

Задача 14
Напишите функцию, которая принимает текст и выводит два слова: наиболее часто встречающееся и самое длинное.
